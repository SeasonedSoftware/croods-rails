## Installation
Add this line to your application's Gemfile:

```ruby
gem 'croods'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install croods
```


## Contributing

You can manually check your changes in the dummy Rails app under `/todos`.

To set it up:
```
cd todos/
bin/setup
```

To run specs use:
```bin/rspec```
