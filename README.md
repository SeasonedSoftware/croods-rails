## Installation
Add this line to your application's Gemfile:

```ruby
gem 'croods'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install croods
```

## Contributing
There's a ~not so~ dummy Rails app in `/todos` directory.

You can use it to test your changes.

To set it up:
```
cd todos/
bin/setup
```
